Bootstrap

Info

Ramverket Bootstrap har till uppgift att underlätta och snabba på utvecklandet av design för webbplatser. Bootstrap, till skillnad från Tailwind, innehåller mer "färdiga" komponenter som gör det enklare och snabbare, men svårare att anpassa teman och i vissa fall layout.

Utvärdering

Jag tyckte Bootstrap var snabbt och smidigt, samt enklare än Tailwind. Fördelen med Bootstrap var att det var enkelt att komma igång med och att de "färdiga" komponenterna som cards och tables gjorde arbetet enkelt. Eftersom raderna i tabellen skrivs ut med JavaScript och att jag ville kunna återanvända JS-filen så mycket som möjligt, var det skönt att enbart behöva sätta styling på table-elementet. En annan fördel var att jag inte behövde skriva egen JS kod för navigeringsmenyn. Nackdelen med Bootstrap är att den ger för mycket och är bättre för mindre projekt eller där designen bara ska fungera.
