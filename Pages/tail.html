Tailwind

Info

Ramverket Tailwind ska kunna snabba på designutvecklingen för en webbplats. Tailwind är mycket anpassningsbart och har tekniker för att minska upprepning.

Utvärdering

Jag tyckte det var trevligt att arbeta med Tailwind, det kanske inte var snabbt som Boostrap, men jag kände att jag hade mer kontroll. Den största fördelen var användandet av @layer base/components med @apply. Jag kunde definiera min .custom-rows klass och ge den Tailwinds fördefinierade klasser med @apply. Detta var för att raderna i tabellen skrevs ut med JavaScript och att jag ville kunna återanvända JS-filen så mycket som möjligt. Nackdelen med Tailwind var att jag behövde skriva min egna JS-kod för navigeringsmenyn. En annan var att Tailwind inte hade något som liknade Bootstrap was-validated klass som ger en styling på formuläret med ikoner och färg.
